# Excel管理・加工チェックツール

## 概要

Excelで管理されている管理表を対象に、
指定ルールに基づいた情報の転記・版数更新・差分確認を支援する業務向けGUIツールです。

日常業務で発生しやすい「転記ミス」「確認漏れ」「手作業による工数増大」を削減することや
マスターデータ更新時の変更点確認や、誤更新の検知を目的として作成しました。

※ 本ツールは業務利用を想定しているため、実装コードは公開していません。

## 作成背景

* Excelシート構成が複雑で、更新時の転記作業に時間がかかっていた
* 手作業によるコピー＆ペーストでミスが発生しやすかった
* 加工前後の差分確認に手間がかかっていた

これらの課題を解消するため、
**人が判断すべき部分と機械に任せる部分を分離した支援ツール**として設計しました。

---

## 主な機能

* 加工対象シート・列をGUI上で選択
* 版数・開発名・チェックサム等の自動転記
* シート構成に応じた例外ルールへの対応
* 加工後ファイルの自動リネーム・更新日時設定
* 加工前後の差分表示（加工あり／加工なし比較）
* ドラッグ＆ドロップによるファイル指定

---

## 技術構成

* Python
* Tkinter（GUI）
* openpyxl（Excel操作）
* multiprocessing（処理分離）

---

## 工夫点

* 結合セルを考慮したコピー処理により、Excel構造依存の不具合を防止
* シート構成や運用ルールの違いに対応できる柔軟な処理設計
* GUIによる選択式操作で、誤操作を防止
* OneDrive環境でのファイルロックを考慮した安全な保存方式
